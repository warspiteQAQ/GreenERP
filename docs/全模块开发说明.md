# GLDG ERP 全模块开发说明

> 统一结构：标题 -> 模块功能描述 -> 各按钮及简单功能描述 -> 表结构（字段名称、字段类型、存储数据、是否可以为空）

## 1. 登录与系统框架模块

### 标题
- 登录与系统框架

### 模块功能描述
- 数据来源：`users`、`roles`、`user_roles`、`permissions`、`role_permissions` 等权限基础数据。
- 数据处理过程：完成登录入口、主窗口装载、左侧菜单导航和页面切换。
- 数据流向：认证后将用户上下文与导航控制流向各业务模块页面。

### 按钮及简单功能描述
- 登录：当前为演示模式（代码中登录校验已短路）。
- 左侧导航按钮：切换报价、项目、订单、询价、采购、生产、仓库、财务、物料、用户、角色模块。
- 侧边栏折叠按钮（`<` / `>`）：展开或收起左侧导航栏。

### 表结构
- 本模块未新增独立业务表，主要依赖用户权限相关表：`users`、`roles`、`user_roles`、`permissions`、`role_permissions`。

## 2. 报价管理模块

### 标题
- 报价管理

### 模块功能描述
- 数据来源：当前页面为占位实现，未接入报价业务数据源。
- 数据处理过程：仅提供页面刷新与占位展示，未执行业务落库。
- 数据流向：后续可流向询价/采购链路；当前无实际数据流。

### 按钮及简单功能描述
- 刷新：刷新当前占位页面文案。

### 表结构
- 当前页面为占位实现，暂未直接落库。
- 实际询报价流程落在“询价管理模块”相关表（见第 5 节）。

## 3. 项目管理模块

### 标题
- 项目管理

### 模块功能描述
- 数据来源：项目主数据由本模块直接维护（`projects`）。
- 数据处理过程：录入/编辑项目基础信息并按状态、关键词筛选展示。
- 数据流向：项目数据流向订单管理模块，作为订单创建的上游来源。

### 按钮及简单功能描述
- 新建：新增项目。
- 编辑：编辑选中项目。
- 刷新：刷新项目列表。
- 状态筛选下拉框：按项目状态过滤（如新建/进行中/已完成/已取消）。
- 关键词输入框：按项目编码/名称/客户查询。

## 4. 订单管理模块

### 标题
- 订单管理

### 模块功能描述
- 数据来源：订单数据来源于项目管理模块（`projects`），物料来源于物料管理模块（`materials`）。
- 数据处理过程：基于项目信息创建订单（`orders`），并录入订单物料明细（`order_details`）。
- 数据流向：订单数据流向询价、采购、生产模块，作为后续业务驱动源。

- 编辑：编辑选中订单。
- 刷新：刷新订单与明细。

- 新建物料：新建物料后回填到订单明细。
- 编辑明细：修改明细数量、单价。
- 删除明细：删除选中订单明细。

## 5. 询价管理模块

### 标题
- 询价管理

### 模块功能描述
- 数据来源：数据来源于订单管理模块（`orders`、`order_details`）和物料/BOM数据（`materials`、`material_components`）。
- 数据处理过程：从订单生成询价单与询价明细，录入供应商报价并执行中标选择。
- 数据流向：中标供应商与中标价格流向采购管理模块，支持采购单定价与供应商选择。

### 按钮及简单功能描述
- 查询：按询价单号/来源订单查询。
- 订单询价：从订单自动生成询价单与询价明细。
- 刷新：刷新询价数据。
- 录入信息：录入或修改供应商报价信息。
- 自动选最低价：按价格（及交期）自动选择中标报价。
- 刷新报价：刷新当前物料报价列表。
- 右键“强制选中该条目”：手动指定中标报价。

## 6. 采购管理模块

### 模块功能描述
- 数据来源：采购需求来源于订单管理模块，价格和供应商可来源于询价管理中标结果。

## 7. 生产管理模块

### 标题
- 生产管理

### 模块功能描述
- 数据来源：生产任务来源于订单管理中的生产件明细，BOM来源于物料管理。
- 数据处理过程：订单转生产单，展开组件用料，执行领料、开工、完工并生成生产出入库单。
- 数据流向：领料/完工单据数据流向仓库管理模块，驱动库存出入与流水。

### 按钮及简单功能描述
- 查询：按生产单号/来源订单查询。

- 刷新：刷新生产单及明细。
- 明细右键“生产领料”：生成生产出库单（`production_out`）。
- 明细右键“开始生产”：校验领料单已完成后开工。
- 明细右键“生产完成”：完工并自动生成生产入库单（`production_in`）。

## 8. 仓库管理模块

### 标题
- 仓库管理

### 模块功能描述
- 数据来源：来源于采购到货、生产领料/完工、手工出入库单据数据。
- 数据处理过程：维护仓库/库位，执行入库、出库、过账，更新库存余额并写入库存流水。
- 数据流向：库存结果回流支撑采购、生产、订单等模块的库存可用性与执行状态。

### 按钮及简单功能描述
- 库存总览页：新增仓库、新增库位、刷新库存。
- 出入库页：新增单据、新增明细、删除明细、过账、刷新单据。
- 出入库页右键单据：完成入库 / 完成出库。
- 库存流水页：刷新流水。
